# uf-autorespawn-with-spawnkill

this plugin is autorespawn with spawnkill detected. 
most used for course servers like mine.


// How to add into your server:
1: download latest update plugin.
2: Upload addons into your server ftp.
3: restart your server.
4: Go to addons/counterstrikesharp/configs/plugins/uf-autorespawn
5: edit: uf-autorespawn.json
6: Change respawn time if you need it.
7: save file.
8: change map or restart server.
9: and your done.
